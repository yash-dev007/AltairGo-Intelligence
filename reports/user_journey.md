# User Side Plan (The Journey)

This document outlines the experience of a traveller using the **AltairLabs Travel Intelligence Platform**.

## Phase 1: Inspiration & Discovery
**Goal**: Catch the user's attention and inspire them to explore.
- **Landing Page**: User arrives at a modern, visually immersive Hero section.
- **Browse Destinations**: User scrolls through "Trending Destinations" (Carousel) to see top spots like El Nido or Kyoto.
- **Search**: User uses the Search Bar to find specific places (e.g., "Paris").
- **Explore Packages**: User views curated travel packages for hassle-free booking.

## Phase 2: Planning & Intelligence
**Goal**: Empower the user with data-driven planning tools.
- **Trip Planner (AI)**: 
    1.  User clicks "Plan your trip".
    2.  Selects Country (e.g., Japan).
    3.  Selects Dates and Preferences (Budget, Crowd Level).
- **Smart Insights**: The system warns about high crowd levels or high budgets in real-time, offering tips like "Book early morning tours".
- **Itinerary Generation**: User receives a detailed day-by-day itinerary based on their selections.

## Phase 3: Deep Dive & Decision
**Goal**: Provide all necessary details to make a decision.
- **Destination Details**: User clicks on a destination card.
    - **Visuals**: Sees high-quality images and highlights.
    - **Information**: Reads about the location, best time to visit, and costs.
    - **Social Proof**: Reads reviews from other travelers to verify quality.

## Phase 4: Interaction (Reviews)
**Goal**: build community and trust.
- **Reading Reviews**: User sorts reviews by "Newest" or "Highest Rated" and checks the Rating Distribution (e.g., how many 5-star ratings).
- **Writing Reviews**:
    1.  User clicks "Write a Review".
    2.  Fills out Name, Rating (1-5 stars), and Review Text.
    3.  Submits -> Review appears instantly on the page via Optimistic UI.
